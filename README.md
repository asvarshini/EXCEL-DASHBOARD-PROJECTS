# Sales Performance Dashboard

An interactive Excel dashboard built using Microsoft Excel, Pivot Tables, Charts, Slicers, and VBA.

## Features
- Dynamic filtering using slicers
- VBA-based dashboard interaction
- Interactive charts and visualizations
- Sales performance analysis
- Region-wise filtering and insights

## Tools & Technologies
- Microsoft Excel
- VBA
- Pivot Tables
- Pivot Charts
- Data Visualization
